===================================== 
  달리기 기록 저장 프로그램 RunTo 안내
=====================================

본 프로그램은 안드로이드 어플리케이션으로 안드로이드 기기에서만 실행이 가능하다. 
프로그램 실행 요구 사항 및 개발 언어에 대한 정보는 다음과 같다.

- 본 프로젝트의 어플리케이션은 Andrpod Tiramisu(API level 33) 버전부터 지원한다. 
- 개발 언어는 Kotlin으로 사용한다

다운로드 방법은 다음과 같다
1. RunTo.apk 파일을 다운로드 한다
  플레이스토어에서 다운로드 하는 것이 아니기 때문에 신뢰할 수 있는 출처에서만 파일을 다운로드 하라는 안내 문구가 뜬다.
  이를 무시하고 다운로드를 클릭한다.  
  만약 보안을 위해 알 수 없는 출처... 와 같은 안내 문구가 뜨면서 설치가 되지 않는다면 설정->어플리케이션->특별한 접근->출처를 알 수 없는 앱 설치에서 설치하는 경로의 앱을 허용 상태로 만든 후 다시 다운로드한다. 
2. 다운로드가 완료되면 설치를 진행한다
  만약 설치가 되지 않는다면 내 파일에 들어가 RunTo.apk 파일을 클릭하여 설치한다.
  안전하지 않은 앱 차단됨이라는 팝업이 뜬다면 아래의 세부정보 더보기를 클릭하여 무시하고 설치하기를 누른다. 
3. 앱이 설치되었다는 문구가 뜨면 RunTo 앱을 실행시킨다.
